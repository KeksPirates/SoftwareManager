API Documentation

This API allows you to search for posts on rutracker.

API URL

https://api.michijackson.xyz/


Endpoint

GET/search?q={query}

Parameter

Type

Required

Description

q

String

True

Search query

Example Request

GET api.michijackson.xyz/search?q=keks
